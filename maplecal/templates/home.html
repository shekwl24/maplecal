메이플 계산기

메이플을 더 재밌게 플레이하기 위해 개인이 만든 사이트입니다.

모바일도 구동하나 PC에 최적화되었습니다.

최근 업데이트 사항(5/3)

  • 심볼 계산기 닉네임 검색 기능 추가
  • 깃허브에서 개인 서버로 이전
  • 현재 서버가 많이 불안정합니다.
    다운 되거나 검색 기능 실행안될 수 있음

당분 간 서버 안정화 작업 제외 업데이트는 없을 예정입니다.
